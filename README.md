# RugPullBoss Airdrop 🚨

Bem-vindo ao repositório oficial do **RugPullBoss**, a plataforma definitiva para distribuir tokens "grátis" antes que a liquidez desapareça! Agora com suporte multi-chain para que possamos enganar... digo, alcançar mais investidores desavisados! 💰💀

## Características 💎

- **Suporte para múltiplas redes blockchain** (ERC20, BEP20, EVM... porque um golpe só não basta!)
- **Sistema de referral**: traga seus amigos para a festa antes que tudo vá pelos ares! 🎉
- **Interface amigável**: para facilitar a navegação enquanto o FOMO consome sua alma. 🫠
- **Integração com carteiras Web3** (MetaMask, etc.) para garantir que ninguém escape do rug pull. 🏃‍♂️
- **Contrato inteligente em Solidity**: projetado para parecer legítimo enquanto esvaziamos o caixa. 😈

## Pré-requisitos ⚠️

- Muita fé no projeto (ou um bom advogado 🏛️)
- Node.js (v14 ou superior, caso você queira "conferir" o código)
- npm (v6 ou superior, porque o golpe é moderno!)
- Git (para clonar antes de ser clonado)
- MetaMask ou outra carteira Web3 (vai facilitar sua doação involuntária de fundos)

## Instalação e Configuração 🔧

1. Clone o repositório (antes que sumamos com ele):
   ```sh
   git clone https://github.com/rugpullboss/airdrop.git
   cd rugpullboss-airdrop
   ```

2. Instale as dependências suspeitas:
   ```sh
   npm install --legacy-peer-deps
   ```
   ```sh
   npm install --force  # Confie, é seguro! (Ou não...)
   ```

3. Configure suas variáveis de ambiente obscuras:
   Crie um arquivo `.env.local` e preencha com as seguintes informações (falsas ou não, depende de você):
   ```sh
   NEXT_PUBLIC_CONTRACT_ADDRESS_ETHEREUM=endereco_do_golpe
   NEXT_PUBLIC_CONTRACT_ADDRESS_BSC=outro_endereco_duvidoso
   NEXT_PUBLIC_CONTRACT_ADDRESS_POLYGON=fique_atento
   NEXT_PUBLIC_INFURA_ID=boa_sorte
   ```

4. Atualize os arquivos essenciais para parecer um projeto sério:
   - `hooks/use-web3.ts`: Engane-se mais um pouco.
   - `lib/airdrop.ts`: Ajuste as "regras" do jogo.
   - `app/page.tsx`: Personalize sua armadilha.
   - `components/airdrop-stats.tsx`: Faça parecer que tudo está indo bem.
   - `components/timeline.tsx`: Mostre um cronograma falso para parecer legítimo.

5. Implante o contrato "inteligente":
   - Use Remix IDE ou Truffle (ou qualquer ferramenta que faça parecer profissional) para lançar o contrato **RugPullBossAirdrop.sol**.
   - Atualize os endereços no arquivo `.env.local` e faça sua última oração. 🙏

6. Execute o projeto localmente:
   ```sh
   npm run dev
   ```

7. Acesse `http://localhost:3000` e veja a mágica (ou tragédia) acontecer. 🪄

## Implantação 🚀

Quer espalhar o caos em grande escala? Use a Vercel:

1. Faça push do seu repositório no GitHub (antes de desativá-lo misteriosamente).
2. Conecte-o à Vercel e tente não rir.
3. Configure as variáveis de ambiente como se fossem reais.
4. Implante o projeto e assista o dinheiro fluir (por pouco tempo). 💸
